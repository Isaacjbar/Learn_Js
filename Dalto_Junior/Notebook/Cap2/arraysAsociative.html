12) Arrays Asociativos

Definición, Sintaxis y Usos

Definición

Un array asociativo es una estructura de datos que permite asociar claves (keys) con valores (values).

A diferencia de los arrays normales, en los arrays asociativos las claves pueden ser cadenas de texto en lugar de índices numéricos.

Esto facilita el acceso a los valores mediante nombres descriptivos en lugar de índices numéricos.

Sintaxis y Usos

La sintaxis de un array asociativo en JavaScript es similar a la de un objeto literal:

let arrayAsociativo = {
    clave1: valor1,
    clave2: valor2,
    clave3: valor3
};

Los arrays asociativos son útiles para almacenar datos relacionados entre sí y acceder a ellos de manera más intuitiva mediante nombres descriptivos.

Suelen ser utilizados para representar registros de una base de datos, configuraciones de aplicaciones, opciones de configuración, etc.
