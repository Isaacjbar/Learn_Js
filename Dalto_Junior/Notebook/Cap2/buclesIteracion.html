13) Bucles e Iteración

Definición y Usos

sentencia while

La sentencia while crea un bucle que ejecuta una sentencia mientras la condición especificada sea evaluada como verdadera.

Se utiliza cuando no se sabe de antemano cuántas veces se repetirá el bucle.

while (condición) {
    // código a ejecutar
}

sentencia do while

La sentencia do while crea un bucle que ejecuta una sentencia una vez y luego se repite mientras una condición especificada sea verdadera.

Se asegura de que el código dentro del bucle se ejecute al menos una vez, incluso si la condición es falsa desde el principio.

do {
    // código a ejecutar
} while (condición);

sentencia for

La sentencia for crea un bucle que se repite hasta que una condición especificada sea falsa.

Es comúnmente utilizado cuando se sabe de antemano cuántas veces se repetirá el bucle.

for (inicialización; condición; actualización) {
    // código a ejecutar
}

sentencia for in

La sentencia for in itera sobre las propiedades enumerables de un objeto.

Es útil cuando se desea iterar sobre las claves de un objeto.

for (variable in objeto) {
    // código a ejecutar
}

sentencia for of

La sentencia for of itera sobre los valores de un objeto iterable, como un array o una cadena de texto.

Es útil cuando se desea iterar sobre los valores de un objeto iterable.

for (variable of iterable) {
    // código a ejecutar
}

sentencia break

La sentencia break termina la ejecución del bucle actual, del switch, o de la etiqueta asociada más cercana.

Se utiliza para salir prematuramente de un bucle.

break;

sentencia label

La sentencia label identifica un bucle y le permite usar la sentencia break o continue desde dentro del bucle anidado.

Es útil cuando se desea romper o continuar un bucle específico dentro de bucles anidados.

nombreDeEtiqueta: while (condición) {
    // código a ejecutar
}

sentencia continue

La sentencia continue salta una iteración en el bucle y continúa con la siguiente iteración.

Se utiliza para omitir el resto del código en una iteración y continuar con la siguiente.

continue;
