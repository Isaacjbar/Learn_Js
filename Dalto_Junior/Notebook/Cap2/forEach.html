15) forEach

Sintaxis y uso

Sintaxis y uso

El método forEach ejecuta una función dada una vez por cada elemento de un array.

array.forEach(function(elemento, índice, arreglo) {
    // código a ejecutar
});

El parámetro elemento representa el elemento actual del array.

El parámetro índice representa el índice del elemento actual del array.

El parámetro arreglo representa el array al que pertenece el elemento actual.

Ventajas respecto a for

1. Sintaxis más simple y legible.

2. No es necesario preocuparse por el tamaño del array ni por el manejo de índices.

3. La función de callback proporciona acceso directo a cada elemento sin necesidad de acceder a través de un índice.

Desventajas (Break, Continue)

1. No es posible utilizar break para salir del bucle forEach.

2. No es posible utilizar continue para omitir una iteración en el bucle forEach.

Para salir o omitir iteraciones en un bucle forEach, se recomienda usar un bucle for o for...of.
