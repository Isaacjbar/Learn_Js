11) Arrays

Definición y Usos

Definición

Un array es una estructura de datos que puede almacenar múltiples valores bajo un mismo nombre.

Se utilizan para almacenar colecciones de datos, como listas de elementos o conjuntos de información relacionada.

Los arrays son especialmente útiles para trabajar con conjuntos de datos homogéneos, es decir, datos del mismo tipo.

Usos

Los arrays son utilizados en programación para manejar colecciones de elementos de manera eficiente.

Se utilizan en situaciones donde se necesita almacenar y acceder a múltiples valores de manera organizada.

Algunos ejemplos de usos comunes de los arrays incluyen almacenar listas de números, cadenas de texto, objetos, etc.

Formas de crear un array

  • Declaración directa: `let array = [];`
  • Con el constructor Array: `let array = new Array();`
  • Con valores iniciales: `let array = [valor1, valor2, valor3];`
  • Mediante el método push(): `array.push(valor);`
