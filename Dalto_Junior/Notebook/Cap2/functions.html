14) Funciones

Definición y Usos

Definición y Usos

Una función en programación es un bloque de código que realiza una tarea específica cuando es llamado o invocado.

Las funciones permiten modularizar el código, reutilizarlo y hacerlo más legible y mantenible.

Se utilizan para dividir un programa en pequeñas partes, cada una realizando una tarea específica.

Formas de crear funciones (y llamarlas)

1. Declaración de función:

function nombreFuncion(parametros) {
    // código a ejecutar
}

2. Expresión de función:

let nombreFuncion = function(parametros) {
    // código a ejecutar
};

Para llamar una función:

nombreFuncion(argumentos);

Return

La sentencia return finaliza la ejecución de la función y especifica el valor que se debe devolver al lugar desde donde fue llamada.

function sumar(a, b) {
    return a + b;
}
let resultado = sumar(5, 3); // resultado será 8

Parámetros

Los parámetros son variables que se utilizan para pasar datos a una función cuando es llamada.

function saludar(nombre) {
    console.log("Hola, " + nombre + "!");
}
saludar("Juan"); // Salida: Hola, Juan!

Scope

El scope o ámbito de una variable es la parte del programa donde esa variable es accesible.

Las variables declaradas dentro de una función tienen un scope local, lo que significa que solo pueden ser accedidas desde dentro de la función.

function ejemplo() {
    let variableLocal = "Soy local";
    console.log(variableLocal); // Salida: Soy local
}
console.log(variableLocal); // Error: variableLocal no está definida

Funciones flecha

Las funciones flecha son una forma más concisa de escribir funciones en JavaScript.

Son especialmente útiles para funciones simples y funciones anónimas.

// Sintaxis básica
let nombreFuncion = (parametros) => {
    // código a ejecutar
};
// Ejemplo
let sumar = (a, b) => a + b;
