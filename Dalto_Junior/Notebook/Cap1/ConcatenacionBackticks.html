Concatenación

Definición: La concatenación es el proceso de unir dos o más cadenas de texto. En JavaScript, se puede realizar de varias maneras.

Ejemplo: Unir dos cadenas "Hola" y "Mundo" para formar "Hola Mundo".

Con el signo +: let saludo = "Hola" + " Mundo";

Para números, forzamos a string con (""): let numeroTexto = 5 + "" + 10; // "510"

Con concat: let saludo = "Hola".concat(" Mundo");

Con backticks y la variable entre ${}: let nombre = "Mundo"; let saludo = `Hola ${nombre}`;

Uso de Backticks

Definición: Los backticks (acento grave) se utilizan para definir plantillas de cadena (template literals) en JavaScript, permitiendo cadenas multilínea y expresiones incrustadas.

Sin backticks: let texto = "línea 1\n\ línea 2";

Con backticks: let texto = `línea 1 línea 2`;

Escape de comillas simples: let texto = `Es un 'texto' con comillas simples`;

Escape de comillas dobles: let texto = `Es un "texto" con comillas dobles`;
