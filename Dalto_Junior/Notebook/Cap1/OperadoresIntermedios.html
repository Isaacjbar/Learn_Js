Operadores en JavaScript

Fecha: 04/06/2024

Notas

  • Operadores Lógicos
  • Operadores de Comparación

Detalles

Operadores Lógicos

  • && (AND lógico): Devuelve true si ambos operandos son true.
    let resultado = true && false; // false
  • || (OR lógico): Devuelve true si al menos uno de los operandos es true.
    let resultado = true || false; // true
  • ! (NOT lógico): Devuelve true si el operando es false, y false si el operando es true.
    let resultado = !true; // false

Operadores de Comparación

  • == (igual a): Compara dos valores para igualdad después de convertirlos a un tipo común.
    let resultado = 5 == '5'; // true
  • === (estrictamente igual a): Compara dos valores para igualdad sin realizar conversión de tipo.
    let resultado = 5 === '5'; // false
  • != (diferente de): Compara dos valores para desigualdad después de convertirlos a un tipo común.
    let resultado = 5 != '5'; // false
  • !== (estrictamente diferente de): Compara dos valores para desigualdad sin realizar conversión de tipo.
    let resultado = 5 !== '5'; // true
  • > (mayor que): Devuelve true si el operando izquierdo es mayor que el derecho.
    let resultado = 5 > 3; // true
  • < (menor que): Devuelve true si el operando izquierdo es menor que el derecho.
    let resultado = 5 < 3; // false
  • >= (mayor o igual que): Devuelve true si el operando izquierdo es mayor o igual que el derecho.
    let resultado = 5 >= 3; // true
  • <= (menor o igual que): Devuelve true si el operando izquierdo es menor o igual que el derecho.
    let resultado = 5 <= 3; // false

Resumen

Los operadores lógicos se usan para realizar operaciones booleanas, mientras que los operadores de comparación se usan para comparar dos valores y devolver un resultado booleano basado en esa comparación.
